Station Tool

Many teachers use stations teaching as a way to teach different activities by rotating students to different stations. Use this tool to drag and drop different tools that students will need to bring to their station. Click the "Rotate stations!" button when it is time to rotate! Don't forget to input the names of students in each station so students know where to go. Enjoy!
